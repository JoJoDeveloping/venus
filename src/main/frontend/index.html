Registers Pattern:

To add tabs, type '\t' & new line, type '\n'. The current valid symbols to add are:
%0% through %31% which represent the registers.
%line% is the current line which it is on.
%pc% represents the pc at the current instruction.
%inst% represents the current instruction.
%output% represents an ecall message out.
%decode% represents the decoded instruction bits.
(⇓ Green = True; White = false ⇓)
